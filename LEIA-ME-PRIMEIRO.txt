COMO INICIAR (ESCOLHA UMA OPÇÃO):
================================================================================

OPÇÃO 1 - MAIS FÁCIL:
  1. Clique duas vezes no arquivo: iniciar.bat
  2. Aguarde o navegador abrir automaticamente
  3. Se não abrir, acesse: http://localhost:5173

OPÇÃO 2 - MANUAL:
  1. Abra o terminal/PowerShell nesta pasta
  2. Digite: npm run dev
  3. Acesse: http://localhost:5173 no navegador

================================================================================
⚠️ IMPORTANTE:
================================================================================

- NÃO abra o arquivo index.html diretamente no navegador
- O servidor DEVE estar rodando (npm run dev)
- Use sempre o endereço http://localhost:5173
